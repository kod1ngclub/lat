# lat.css

"Line and text"

## Philosophy

- anti-colorful
- only 'shadow' color
- lightweight

## Getting Started

```bash
# install package
npm install -g sass

# clone git
git clone https://github.com/kod1ngclub/lat.git
cd lat

# build
sass --style=compressed index.scss build.min.css

echo "⛏️ build.min.css"
```

## How to use

We don't make additional docs.
Instead, you can learn how to use by `examples`
